Difference between var and lin

Var

  • Var is function scope
  • var can be accessed outside the looop
  • It acts as global variable
  • It can be updated and redeclared within its scope
  • It can be initialized without declaring

    • Let

  • It is limited to block defined by {}
  • Let can be accessed inside the scope it is declared in loop
  • It acts as local variable
  • It can be updated but not redeclared
  • It cannot be initialized without declaring

    • Const

  • It is limited to block defined by {}
  • It cannot be reassigned
  • It can neither be redeclared nor updated
  • It cannot be initialized without declaring
  • It acts as local variable
